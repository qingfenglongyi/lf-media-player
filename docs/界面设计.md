# 播放器界面

居中显示，宽度和高度自动适应屏幕，自动适应横屏和竖屏，不能过高或过宽出现滚动条。
上下左右各留 10px 的边界。

从上到下分别是：标题区、信息区、进度条、控制区，之间有 20px 间距。

## 标题区
- 显示：歌曲标题 - 艺术家名称
- 高度：50px，居上，左右居中

## 信息区
- 高度：自适应（填充剩余空间），左右居中
- 显示：转盘或歌词，通过点击切换
- 转盘区：圆形，旋转动画，居中，点击切换到歌词
- 歌词区：长方形，充满整个信息区，不显示滚动条，点击切换到转盘

## 进度条
- 高度：20px，居下，左右居中
- 左侧显示当前时间，右侧显示总时长

## 控制区
- 高度：40px（横屏）/ 80px（竖屏两行布局），居下，左右居中
- 横屏布局：一行，按键大小相同，从左到右：播放模式、上一曲、播放/暂停、下一曲、列表
- 竖屏布局：两行，上行（上一曲、播放/暂停、下一曲，按键较大），下行（播放模式、列表，按键略小）
- 按键：播放模式(🔁/🔂/🔀)、上一曲(⏮)、播放/暂停(⏯/⏸)、下一曲(⏭)、列表(📋)


# 列表面板

遮盖整个播放器界面显示。
宽度和高度占满整个屏幕，从右侧滑入显示。
在顶端显示"播放列表""歌单""本地歌曲库"3个 tab 页面。

## Tab 标签栏
- 位置：面板顶部
- 高度：48px
- 布局：三个标签均匀分布（SpaceEvenly）
- 标签文字：播放列表、歌单、本地歌曲库
- 选中状态：青色(#00D4AA)文字+粗体
- 未选中状态：灰色文字+普通字重
- 关闭按钮：位于右上角，✕图标

## 播放列表 Tab
- 显示当前正在播放的歌曲列表
- 歌曲项布局：
  - 左侧：复选框（选中状态显示，未选中隐藏）+ 序号（灰色，12sp）
  - 中间：歌曲标题（白色，14sp）+ 艺术家（灰色，12sp）
  - 右侧：时长（灰色，12sp）
- 当前播放歌曲：背景高亮(#2A2A4E)，序号和标题显示青色
- 支持点击切换歌曲
- 多选模式（点击"选择"按钮激活）：
  - 歌曲列表左侧显示复选框
  - 右上角显示"选择/取消选择"按钮（始终显示）和菜单按钮（📋，始终显示）
  - 点击菜单按钮弹出下拉菜单，菜单项目：
    - 清空播放列表（始终显示）
    - 删除（至少选中一首歌曲后显示）
    - 添加到歌单（至少选中一首歌曲后显示）
  - 选中至少一首歌曲后，菜单里的"删除"和"添加到歌单"才显示
  - 点击"选择/取消选择"按钮，可激活或取消多选模式

## 歌单 Tab
- 布局：分为歌单列表和创建按钮两部分
- 创建歌单按钮：
  - 布局：📋图标 + "创建新歌单"文字（青色）
  - 位置：列表顶部
- 歌单列表项：
  - 左侧：📋图标
  - 中间：歌单名称
  - 右侧：🗑删除图标
- 点击歌单进入歌单详情：
  - 显示返回按钮(←) + 歌单名称 + 歌曲数量
  - 歌曲列表项：💿图标 + 歌曲标题 + 艺术家 + 时长 + 删除按钮(🗑)
  - 点击删除按钮可将该歌曲从歌单中移除
- 创建/删除歌单：使用对话框操作

## 本地歌曲库 Tab
- 顶部：搜索框（带"搜索歌曲..."提示文字）
- 分类选项栏：
  - 三个筛选标签：全部、艺术家、专辑
  - 选中状态：青色背景圆角胶囊
  - 未选中状态：灰色背景
- 歌曲列表（全部视图）：
  - 与播放列表Tab相同的歌曲项布局和操作方式
  - 多选模式（点击"选择"按钮激活）：
    - 歌曲列表左侧显示复选框
    - 右上角显示"选择/取消选择"按钮（始终显示）和菜单按钮（📋，始终显示）
    - 点击菜单按钮弹出下拉菜单，菜单项目：
      - 添加到播放列表（至少选中一首歌曲后显示）
      - 添加到歌单（至少选中一首歌曲后显示）
      - 从歌曲库中删除（至少选中一首歌曲后显示）
    - 选中至少一首歌曲后，菜单项才显示
    - 点击"选择/取消选择"按钮，可激活或取消多选模式
- 艺术家列表视图：
  - 列表项：🎵图标 + 艺术家名 + 歌曲数量
  - 点击进入艺术家详情页
- 艺术家详情页：
  - 返回按钮(←) + 艺术家名 + 歌曲数量
  - 显示该艺术家的所有歌曲
- 专辑列表视图：
  - 列表项：💿图标 + 专辑名 + 歌曲数量
  - 点击进入专辑详情页
- 专辑详情页：
  - 返回按钮(←) + 专辑名 + 歌曲数量
  - 显示该专辑的所有歌曲
