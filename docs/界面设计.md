# 播放器界面

居中显示，宽度和高度自动适应屏幕，自动适应横屏和竖屏，不能过高或过宽出现滚动条。
上下左右各留 10px 的边界。

从上到下分别是：标题区、信息区、进度条、控制区，之间有 20px 间距。

## 标题区
- 显示：歌曲标题 - 艺术家名称
- 高度：50px，居上，左右居中

## 信息区
- 高度：自适应（填充剩余空间），左右居中
- 显示：转盘或歌词，通过点击切换
- 转盘区：圆形，旋转动画，居中，点击切换到歌词
- 歌词区：长方形，充满整个信息区，不显示滚动条，点击切换到转盘

## 进度条
- 高度：20px，居下，左右居中
- 左侧显示当前时间，右侧显示总时长

## 控制区
- 高度：40px，居下，左右居中
- 包含：上一曲、播放/暂停、下一曲按钮


# 列表

遮盖整个播放器界面显示。
在顶端显示"播放列表""歌单""本地歌曲库"3个 tab 页面。

## 播放列表
显示当前正在播放的歌曲列表，支持点击切换歌曲。

## 歌单
显示用户创建的播放列表，支持创建、编辑、删除歌单。

## 本地歌曲库
扫描设备上的音乐文件，显示所有可用歌曲。
